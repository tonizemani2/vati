# The mission
An AI that predicts the future: **calibrated, falsifiable, pre-consensus forecasts of where scarcity/value migrates** — proven by a scored record. Rent accrues to the binding constraint; the edge is seeing where it moves **before it's priced in**. The value is the human+AI thinking loop; the code is thin scaffolding (two folders, one DB).

# THE BAR — read this twice
Pre-consensus means catching the **faint, granular, early** signal nobody has priced yet.
- **Missing a shift as large as deep learning is total failure, not a caveat.** If the system goes silent on the biggest winner of the decade, it has no edge — it is a hindsight narrator.
- The alpha lives in **small intricacies**: a sub-topic gaining share, a few elite labs pivoting, a technique crossing from field A into field B, a citation graph reorienting. Aggregate counts are the LAST place a signal appears. Read the earliest, finest grain.
- **Recall at the detector; precision at the gate.** Fire generously on many sharp, orthogonal early signals (each passing its own σ test); let the supply-graph + consensus gates reject the false positives. Never buy precision by going blind to the early move.

# The blind spot we diagnosed — do not repeat it
The detector fits ONE robust trend on a window and flags departure from it. Structurally blind two ways:
1. **In-window regime breaks get absorbed into the trend** → it called AI-compute *decelerating* (−11.8σ) at 2017 because the 2012 break was already inside the fit. Fix: **changepoint / slope-of-slope** — fire on a *change in slope*, not departure from one global slope.
2. **One coarse channel (annual counts)** → leading signals (citation velocity, cross-field diffusion, author/talent inflow, structural-hole/bridge betweenness, sub-topic share) are never computed, so deep learning was invisible at 2010. Fix: **multi-channel leading detectors** — the cross-domain-diffusion + structural-hole detectors we still lack.

No overfitting: add channels for *mechanistic* reasons only; never move a signal date; never tune k on the §8 corpus. Precision/specificity MUST hold on the rolling-origin sweep as recall rises — if a channel tanks precision, it's noise, cut it. Recall comes from more sharp instruments, never from lowering k on the blunt one.
